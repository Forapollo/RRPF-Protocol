# Changelog

## v0.2.0

*   Added engine-side capability annotations (`@table`, `@event`)
*   Declarative, static capability contracts
*   No runtime enforcement (introspection-only)
*   Backward compatible

## v0.1.0

*   Initial public release
*   Deterministic request canonicalization and hashing
*   Strict protocol validation
*   Engine contract and fulfillment runner
*   Partial vs hard-fail semantics
*   Provenance and digest-based immutability
*   Payload storage and replay (memory + filesystem)
*   Reference engines and documentation
